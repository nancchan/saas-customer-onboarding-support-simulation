# Day 3 Report: Customer Data Setup

## Purpose

This step creates the base customer dataset used in a simulated SaaS CRM system.

## What Was Built

- Customer table structure
- Sample customer records
- Basic data retrieval query

## Business Context

In real SaaS systems, customer data is stored in databases before being:
- synced to CRM systems
- used in dashboards
- accessed by support teams

## Key Learning

Understanding how structured customer data is created and stored is the foundation for all SaaS data workflows.
